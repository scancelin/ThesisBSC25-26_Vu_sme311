# ThesisBSC25-26_Vu_sme311
This repository contains the Jupyter notebooks used for the experiments in my Bachelor's thesis at Vrije Universiteit Amsterdam.

The notebooks cover:

1. Data preparation
2. Clean baseline
3. Soft prompt tuning
4. Adapter-based debiasing
5. Full fine-tuning
6. Fairness and sustainability evaluation

The Jigsaw Unintended Bias in Toxicity Classification dataset is not included and should be downloaded separately from Kaggle.
